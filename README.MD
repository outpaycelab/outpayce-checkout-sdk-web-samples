# Outpayce Checkout SDK - Samples

This repository contains examples projects on how to integrate the Outpayce Checkout SDK within your application.

## Available Samples

| Sample | Platform | Description |
|--------|----------|-------------|
| [01_basic_integration/html](examples/01_basic_integration/html) | Web | Basic HTML/CSS/JavaScript integration |
| [01_basic_integration/react](examples/01_basic_integration/react) | Web | React application with Vite |
| [01_basic_integration/ios](examples/01_basic_integration/ios) | iOS | Native iOS app with WKWebView |
| [01_basic_integration/react-native](examples/01_basic_integration/react-native) | iOS & Android | React Native mobile app |
| [02_apple_pay_brick_mode](examples/02_apple_pay_brick_mode) | Web | Angular app with Apple Pay integration |

## Pre-requisites

- Get your Outpayce Checkout Integration by contacting our sales team: 
https://outpayce.com/en/forms/contact-sales
- Be able to use our Payment Checkout Experience API: https://developers.amadeus.com/functional-doc-rest/103
- Be able to generate a PPID

## How to use this repository

Each example is located in the `examples` folder.
For each one, you have a README file explaining how to install and run.
